## 📚 课程概览

ARIN7001 覆盖知识点非常广，常被调侃为“理综”，因为内容涉及：

- 高等数学（导数、梯度、微积分、约束优化等）
- 线性代数（矩阵、特征值、正定性等）
- 概率论与统计（分布、估计、检验等）
- 贝叶斯统计与概率图模型（Bayes rule、BN、条件独立等）
- 数据结构与算法（复杂度、树、堆、遍历等）

---

## 🧠 考试套路

纵观过去三年的期末卷子，会发现出题套路**非常相似**，存在可学习的规律：

- **Math section** 主要考察积分、一共五种类型题目
- **Stat section** 每年基本固定联合分布、最小二乘、贝叶斯网络
- **CS section** 侧重时间空间复杂度计算
- **题型结构高度一致**
- 考察知识点落在一个**可控范围**内

因此本仓库不仅总结知识点，也更强调“**怎么考、怎么拿分**”。
